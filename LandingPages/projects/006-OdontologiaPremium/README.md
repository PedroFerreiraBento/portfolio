# Landing Page: Odontologia Premium

Esta landing page foi desenvolvida para uma clínica odontológica de alto padrão, focando em estética e reabilitação oral. O design utiliza a estética Ultra-Clean High-Tech para transmitir precisão, higiene e confiança.

## 🎨 Estilo e Design

- **Estilo Escolhido**: Ultra-Clean High-Tech.
- **Vibe**: Clínica, moderna, segura e transformadora.
- **Paleta de Cores**:
  - Background: `#FFFFFF` (Branco Puro)
  - Surface: `#F1F5F9` (Slate Ultra Light)
  - Accent: `#2563EB` (Azul Royal Tecnológico)
  - Text: `#1E293B` (Slate Dark)
- **Tipografia**:
  - `Public Sans` para títulos (modernidade e clareza).
  - `Inter` para corpo de texto (legibilidade padrão ouro).

## 🛠️ Tecnologias

- **Astro**: Performance e estrutura de componentes.
- **Tailwind CSS 4**: Design system moderno.
- **daisyUI**: Componentes aceleradores.

## 📈 Estratégia Comercial

A página foca na "Transformação do Sorriso":
1. **Confiança**: Hero com foco no resultado final (sorriso) e tecnologia.
2. **Especialidades**: Apresentação clara dos serviços de alto ticket (Lentes, Implantes).
3. **Tecnologia**: Destaque para equipamentos que reduzem a dor e aumentam a precisão.
4. **Resultados**: Demonstração de excelência técnica.
5. **Ação**: Foco em agendamento de avaliação inicial.
