# Landing Page: Esmalteria Chic

Esta landing page foi desenvolvida para um Nail Bar de alto padrão, utilizando a estética Soft & Chic para transmitir delicadeza, cuidado e luxo acessível.

## 🎨 Estilo e Design

- **Estilo Escolhido**: Soft & Chic (Variação de Clean Premium).
- **Vibe**: Delicada, feminina, boutique e impecável.
- **Paleta de Cores**:
  - Background: `#FFFBFB` (Pó de arroz)
  - Accent: `#E29595` (Dusty Rose)
  - Text: `#2D2D2D` (Cinza Chumbo)
- **Tipografia**:
  - `Cormorant Garamond` para títulos (editorial e sofisticado).
  - `Plus Jakarta Sans` para corpo de texto (moderno e leve).

## 🛠️ Tecnologias

- **Astro**: Performance e SEO local.
- **Tailwind CSS 4**: Design system flexível.
- **daisyUI**: Componentes aceleradores.

## 📈 Estratégia Comercial

A jornada foca no "ritual de beleza":
1. **Atração**: Hero com foco visual na qualidade do serviço.
2. **Especialidades**: Cards visuais mostrando as tendências de Nail Art.
3. **Ambiente**: Reforço da higiene e conforto da boutique.
4. **Comunidade**: Depoimentos e Instagram feed simulado.
5. **Agendamento**: Link direto para WhatsApp facilitado.
