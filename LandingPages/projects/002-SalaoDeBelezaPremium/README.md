# Landing Page: Salão de Beleza Premium

Esta landing page foi desenvolvida para um salão de beleza de alto luxo, utilizando o estilo Clean Premium para transmitir calma, sofisticação e profissionalismo.

## 🎨 Estilo e Design

- **Estilo Escolhido**: Clean Premium / Neo-Minimalista (Referência: [main_styles.md](file:///d:/Projects/Empreender/Portifolio/LandingPages/docs/main_styles.md) - Estilo 1).
- **Vibe**: Sofisticada, organizada, confiável e leve.
- **Paleta de Cores**:
  - Background: `#FAFAF7` (Off-white sofisticado)
  - Accent: `#B88746` (Dourado Queimado)
  - Text: `#151515` (Black-ish)
- **Tipografia**:
  - `Playfair Display` para títulos (presença e elegância).
  - `Inter` para corpo de texto (clareza e modernidade).

## 🛠️ Tecnologias

- **Astro**: Performance estática superior.
- **Tailwind CSS 4**: Estilização baseada em tokens.
- **daisyUI**: Componentes aceleradores.

## 📈 Estratégia Comercial

A página foca na jornada de transformação da cliente:
1. **Desejo**: Hero com imagem aspiracional e promessa de exclusividade.
2. **Serviços**: Apresentação clara de tratamentos premium.
3. **Ambiente**: Galeria que reforça o conforto e o luxo do local.
4. **Confiança**: Depoimentos reais e seção de especialistas.
5. **Conversão**: Agendamento simplificado via WhatsApp.
