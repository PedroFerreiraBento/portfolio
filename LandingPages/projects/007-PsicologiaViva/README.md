# Landing Page: Psicologia Viva

Esta landing page foi desenvolvida para um consultório de psicologia, utilizando a estética Soft Mindful Minimalist para transmitir acolhimento, empatia e clareza mental.

## 🎨 Estilo e Design

- **Estilo Escolhido**: Soft Mindful Minimalist.
- **Vibe**: Zen, acolhedora, humana e equilibrada.
- **Paleta de Cores**:
  - Background: `#FAF7F2` (Warm Sand)
  - Accent: `#84A59D` (Sage Green)
  - Contrast: `#E29578` (Terracotta)
  - Text: `#2D3436` (Charcoal)
- **Tipografia**:
  - `Playfair Display` para títulos (elegância e humanidade).
  - `Plus Jakarta Sans` para corpo de texto (leveza e modernidade).

## 🛠️ Tecnologias

- **Astro**: Performance e estrutura de componentes.
- **Tailwind CSS 4**: Design system flexível.
- **daisyUI**: Componentes aceleradores.

## 📈 Estratégia Comercial

A jornada foca na "Conexão e Alívio":
1. **Acolhimento**: Hero com foco em espaços de respiro e promessa de escuta ativa.
2. **Identificação**: Seção "Sobre" que humaniza o profissional.
3. **Caminho**: Apresentação da abordagem terapêutica de forma simples.
4. **Segurança**: Depoimentos que reforçam o conceito de "espaço seguro".
5. **Ação**: Foco em agendamento de uma conversa inicial sem compromisso.
