# Landing Page: Barbearia Premium

Esta landing page foi desenvolvida para demonstrar um serviço de barbearia de alto padrão, focada em conversão (agendamento via WhatsApp) e estética refinada.

## 🎨 Estilo e Design

- **Estilo Escolhido**: Barbearia Premium (Referência: [design_styles.md](file:///d:/Projects/Empreender/Portifolio/LandingPages/docs/design_styles.md) - Item 121).
- **Vibe**: Dark Mode Premium / Old Money / Vintage.
- **Paleta de Cores**:
  - Background: `#0A0A0A` (Deep Black)
  - Accent: `#D4AF37` (Metallic Gold)
  - Surface: `#1A1A1A` (Dark Grey)
- **Tipografia**:
  - `Playfair Display` para títulos (elegância e tradição).
  - `Inter` para corpo de texto (modernidade e clareza).

## 🛠️ Tecnologias

- **Astro**: Framework para performance estática.
- **Tailwind CSS 4**: Estilização rápida e consistente.
- **daisyUI**: Base de componentes UI.

## 📈 Estratégia Comercial

A página segue uma jornada de venda clássica:
1. **Conscientização**: Hero impactante com proposta de valor clara.
2. **Autoridade**: Trust strip com diferenciais.
3. **Desejo**: Galeria de serviços e fotos de alta qualidade.
4. **Confiança**: Prova social (depoimentos) e FAQ.
5. **Ação**: CTA final chamativo para agendamento.
