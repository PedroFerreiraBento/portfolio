# Landing Page: Clínica Estética Lux

Esta landing page foi desenvolvida para uma clínica de estética médica de alto padrão, utilizando a estética Clean Medical Premium para transmitir autoridade, tecnologia e sofisticação.

## 🎨 Estilo e Design

- **Estilo Escolhido**: Clean Medical Premium.
- **Vibe**: Clínica, tecnológica, confiável e luxuosa.
- **Paleta de Cores**:
  - Background: `#FFFFFF` (Branco Clínico)
  - Surface: `#F8FAFC` (Slate Light)
  - Accent: `#0EA5E9` (Azul Sky / Tecnológico)
  - Text: `#0F172A` (Slate Dark)
- **Tipografia**:
  - `Outfit` para títulos (modernidade e precisão).
  - `Inter` para corpo de texto (legibilidade médica).

## 🛠️ Tecnologias

- **Astro**: Performance e estrutura de componentes.
- **Tailwind CSS 4**: Estilização baseada em tokens.
- **daisyUI**: Componentes aceleradores.

## 📈 Estratégia Comercial

A página foca na "Ciência da Beleza":
1. **Autoridade**: Hero com foco em resultados e profissionalismo médico.
2. **Tecnologia**: Seção estilo Bento Grid destacando equipamentos de última geração.
3. **Tratamentos**: Categorização clara entre faciais, corporais e injetáveis.
4. **Segurança**: Destaque para o corpo clínico e protocolos de segurança.
5. **Conversão**: CTA para avaliação personalizada (Lead Magnet).
