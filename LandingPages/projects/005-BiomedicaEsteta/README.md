# Landing Page: Biomédica Esteta (Personal Branding)

Esta landing page foi desenvolvida para uma Biomédica Esteta de alto padrão, focando em Personal Branding para transmitir autoridade técnica, segurança e um olhar artístico refinado.

## 🎨 Estilo e Design

- **Estilo Escolhido**: Personal Branding Luxury.
- **Vibe**: Sofisticada, acolhedora, técnica e artística.
- **Paleta de Cores**:
  - Background: `#FDFCFB` (Off-white quente)
  - Accent: `#C5A059` (Champagne Gold)
  - Text: `#1A1A1A` (Soft Black)
- **Tipografia**:
  - `Lora` para títulos (elegância e autoridade).
  - `Montserrat` para corpo de texto (modernidade e precisão).

## 🛠️ Tecnologias

- **Astro**: Performance e estrutura de componentes.
- **Tailwind CSS 4**: Design system flexível.
- **daisyUI**: Componentes aceleradores.

## 📈 Estratégia Comercial

A jornada foca na "Confiança na Especialista":
1. **Conexão**: Hero com foco no retrato profissional e promessa de resultados naturais.
2. **Autoridade**: Seção "Sobre" destacando a formação em Biomedicina e o olhar artístico.
3. **Curadoria**: Apresentação dos tratamentos "Assinatura" (queridinhos).
4. **Prova Social**: Resultados reais em galeria minimalista.
5. **Ação**: Conversão direta para agendamento via WhatsApp.
